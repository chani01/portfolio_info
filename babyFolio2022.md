## 사용 스킬
> 클린아키텍처, MVVM, DI, corutine, jetpack compose, roomDB, firebase App Distribution

## 기능 구현
- 홈 화면 개편 1차, 2차
- 영상 업로드 시간 확장
- 키워드 검색 기능
- 미디어 업로드 중복 체크 기능
- 쑥쑥 찰칵 굿즈 상품 제작 기능
- 인 앱 마케팅 툴(쑥톡)
