## 🚀 핵심 기능 구현
**개발 기여도 100%**로 AI 엔진 연동부터 클라이언트 앱 구축까지 전 과정을 전담했습니다.

### 1. AI 기반 꿈 분석 시스템
* **심층 분석 엔진 연동**: AI 모델을 활용하여 사용자가 입력한 꿈의 내용을 분석하고 심리적 해석을 제공하는 핵심 로직 구현
* **백그라운드 분석 프로세스**: 분석 작업이 진행되는 동안 사용자 경험을 저해하지 않도록 백그라운드 처리 로직 설계

### 2. 효율적인 데이터 관리 및 통신
* **GraphQL 도입**: 필요한 데이터만 효율적으로 쿼리하여 네트워크 리소스를 최적화하고 유연한 데이터 통신 환경 구축
* **Cloud Firestore & Room**: 실시간 클라우드 데이터 동기화와 오프라인 환경을 위한 로컬 DB(Room) 연동

### 3. 히스토리 및 사용자 경험
* **꿈 히스토리 리스트**: 과거에 분석한 꿈의 목록을 날짜별/키워드별로 체계적으로 관리할 수 있는 아카이브 기능 구현
* **선언형 UI 구현**: Jetpack Compose를 전면 도입하여 매끄럽고 현대적인 사용자 인터페이스 구축
